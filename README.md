# QCIF Intro to R Shiny

This lesson is a template lesson that uses [The Carpentries Workbench][workbench].

## Overview

This repository contains the teaching materials for QCIF’s introductory R Shiny workshop, with a focus on building interactive web applications using R.

The course is designed for researchers and analysts who:
- already use R for data analysis,
- are new (or relatively new) to Shiny, and
- want to understand how Shiny apps are structured and how data flows reactively through an app.

The lesson follows a hands‑on, live‑coding approach, where participants build a Shiny application step‑by‑step over the course of the workshop.

## Audience
This course assumes participants:
- are comfortable working in R and RStudio,
- have basic familiarity with data frames and R functions,
- have no prior experience with Shiny.

## Learning objectives

By the end of the workshop, participants should be able to:
- Describe the structure of a basic Shiny app (UI and server)
- Add inputs and outputs to control app behaviour
- Explain how reactivity works in Shiny
- Connect data to reactive outputs
